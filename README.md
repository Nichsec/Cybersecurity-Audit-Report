# Cybersecurity Audit Report

## Overview

This project presents a cybersecurity audit conducted for a fictional small business. The assessment evaluates existing security controls, identifies potential risks, and provides recommendations based on cybersecurity best practices and the NIST Cybersecurity Framework (CSF).

## Objectives

- Evaluate the organization's current security posture.
- Identify security risks and vulnerabilities.
- Review existing administrative, technical, and physical controls.
- Recommend improvements to strengthen the organization's security.

## Skills Demonstrated

- Risk Assessment
- Security Controls Evaluation
- NIST Cybersecurity Framework (CSF)
- Security Documentation
- Access Control
- Authentication
- Asset Management
- Security Awareness
- Business Risk Analysis
